# Redbelly Network DAO

The official GitHub organisation for the **Redbelly Network DAO Community Task Board**.

Redbelly is a compliant Layer 1 blockchain purpose-built for Real World Assets (RWAs). Every participant is verified. Every transaction is accountable.

---

## What This Org Is For

This organisation hosts all deliverables produced through the DAO Community Task Board pilot programme. Contributors submit their work here. Reviewers audit it here. The community can verify every deliverable linked in the [Public Transparency Ledger](#).

## How to Submit Work

1. Check the [task board](#) on the DAO website for open tasks
2. Read the full task specification
3. Clone or reference the relevant starter kit in the `resources` repo
4. Build your deliverable in your own repository
5. Open a Pull Request in the appropriate category repo using the provided template
6. A reviewer will be assigned within 48 hours

**Questions?** Post in `#taskboard-updates` on Discord or mention `@njay`.
